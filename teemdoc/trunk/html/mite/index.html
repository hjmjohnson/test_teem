Simple Scalar Volume Renderer

Mite is a scalar field volume renderer based on hoover and gage. Hoover provides the framework for a multi-threaded ray-casting volume renderer, and gage implements the filtered point-sampling that generates the information that is fed into the transfer function. Mite is mainly intended as a proof-of-concept volume renderer. It does very little novel or clever, but its nice to have a high-quality multi-threaded volume renderer that can use separable transfer functions of arbitrarily high dimensions.

Similar to how the unu program is merely a command-line front-end to the unrrdu library, The miter program is a front-end to the mite library. These pages are intended to be documentation for both the miter program and the underlying mite library.
