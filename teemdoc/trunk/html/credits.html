Who helped create Teem
teem

  • Gordon Kindlmann initially created Teem to support his PhD research, and continues to lead development.
  • David Weinstein, J. Dean Brederson, Milan Ikits, Vidya Elangovan, and Joe Kniss, and Aaron Lefohn were early adopters/debuggers.
  • Milan Ikits was the first to build Teem on Windows by engineering a way to use GNU make to create Visual Studio files, helped add Gzip compression for Nrrd, added Bzip2 compression for Nrrd, added support for PNG images to Nrrd, and helps with design decisions.
  • James Bigler wrote/adapted the Mersenne Twister code in Air, wrote the Gzip compression for Nrrd, implemented nrrdTile2D and nrrdUntile2D, and helps with design decisions.
  • David Weinstein was an early adopter/debugger of Nrrd and unu, made Nrrd an basic datatype in SCIRun, and helps with design decisions.
  • Andy Cedilnik added CMake support for Teem and created the Dashboard for Teem.
  • Jonathan Blocksom added support for Mac OSX.
