Known Bugs and Limitations
teem

unu

  • Byte skipping in compressed encodings broken in "unu make"; fixed in teem-1.5beta1.
  • Piping ("|") invocations of unu on Windows doesn't work because of how Windows opens stdin and stdout in ascii mode. This has been fixed in teem-1.5beta1.

Cygwin

There is a (cygwin-imposed) 128 meg limit on the largest chunk of memory that can be allocated. This can potentially effect everything, but may be noticed most with unu and vprobe.
