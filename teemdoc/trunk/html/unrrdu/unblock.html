  unu unblock

unu unblock: Expand "blocks" into scanlines on axis 0. Based on the requested
output type, the number of samples along axis 0 will be determined
automatically. Axis N information will be bumped up to axis N+1. Underlying
data is unchanged.

Usage: unu unblock -t <type> [-bs <blocksize>] [-i <nin>] [-o <nout>]

      -t <type> = type to unblock to
-bs <blocksize> = Useful only if *output* type is also block: the size of
                  blocks in output nrrd (int); default: "0"
       -i <nin> = input nrrd
      -o <nout> = output nrrd (string); default: "-"
