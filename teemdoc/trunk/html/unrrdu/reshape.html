  unu reshape

unu reshape: Superficially change dimension and/or axes sizes. The underlying
linear ordering of the samples is unchanged, but the reported dimension or
axes sizes are changed. Identical in concept to Matlab's "reshape" command.

Usage: unu reshape -s <sz0 sz1  ...> [-i <nin>] [-o <nout>]

 -s <sz0 sz1 ...> = new axes sizes (1 or more ints)
         -i <nin> = input nrrd
        -o <nout> = output nrrd (string); default: "-"
